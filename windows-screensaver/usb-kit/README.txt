APPLEBEE ACRES - screensaver install (per user, no admin needed)

1. Copy this whole folder to the PC (USB stick is easiest).
2. Double-click install.bat
3. Answer Y to see it right away.

Repeat on each PC, logged in as the account that uses it.

- Timer defaults to 3 minutes (change in Settings > Personalization
  > Lock screen > Screen saver, or rerun install.bat after editing it).
- Password on wake is OFF by default (showroom friendly). Tick
  "On resume, display logon screen" in that same Settings page to lock.
- uninstall.bat removes everything.
- Needs 64-bit Windows 10/11 with Microsoft Edge (for WebView2).
- Live weather needs internet access to api.open-meteo.com;
  without it the farm quietly simulates its own weather.
